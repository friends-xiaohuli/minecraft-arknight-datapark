# FORK前的README

## 干员技能获取方法

testworld中有一个用于获取技能的命令方块，修改其中代码，将需要替换的地方替换为技能拼音全称（小写），启动命令方块即可

其他世界若需要使用，需输入指令/function firegreen:arknight/技能拼音全称（小写）/gain
